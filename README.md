# REVO Coffee

Одностраничный сайт о кофейной линейке REVO, сверстанный по предоставленному макету. Страница содержит промо-блок, описание преимуществ кофе, карточки товаров, подарочные наборы и комбо-наборы.

## Технологии

- **HTML5** — структура страницы
- **CSS3** — стилизация, методология БЭМ,  интерактивные состояния
